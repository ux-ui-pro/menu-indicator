### Installation
```
$ yarn add menu-indicator
```

<br>

### Import
```javascript
import MenuIndicator from 'menu-indicator'
```
<br>

### Usage
```HTML
const menuIndicator = new MenuIndicator(
    'nav',
    '.nav__item',
    '.nav__indicator',
    '.active'
)
```
